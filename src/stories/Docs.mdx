### Descrição das Pastas e Arquivos

- **src/**: Diretório principal contendo os arquivos de código-fonte da aplicação.

  - **@types/**: Definições de tipos personalizadas utilizadas no TypeScript.
  - **app/**: Diretório responsável pelo roteamento da aplicação
  - **assets/**: Onde ficam armazenadas as imagens da aplicação.
  - **components/**: Componentes reutilizáveis da interface de usuário.
  - **constants/**: Valores constantes usados ao longo do projeto.
  - **contexts/**: Gerenciamento de estado global usando o Context API.
  - **functions/**: Funções utilitárias.
  - **hooks/**: Hooks personalizados utilizados na aplicação.
  - **http/**: Configurações e serviços relacionados a chamadas de API.
  - **lib/**: Contém alguns objetos de configuração e alguns formatadores.
  - **services/**: Interação com a API externa.

- **middleware.ts**: Configuração do middleware utilizado para trabalhar com rotas privadas.
- **next-env.d.ts**: Configuração gerada automaticamente pelo Next.js para suporte ao TypeScript.
- **next.config.ts**: Configuração personalizada do framework Next.js.
- **package.json**: Contém as dependências do projeto e scripts para gerenciamento.
- **package-lock.json**: Arquivo que define a versão exata das dependências instaladas.
- **pnpm-lock.yaml**: Arquivo de bloqueio para quem utiliza o PNPM como gerenciador de pacotes.
- **postcss.config.mjs**: Configuração do PostCSS para o projeto.
- **README.md**: Documentação do projeto.
- **tailwind.config.ts**: Arquivo de configuração do Tailwind CSS.
- **tsconfig.json**: Configuração do TypeScript.
- **components.json**: Arquivo JSON contendo definições e o mapeamento de componentes da blioteca shadcn ui.
- **.eslint.json**: Configurações para o linting do código usando ESLint.
- **.gitignore**: Especifica arquivos e diretórios a serem ignorados pelo Git.

# Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/ismael-henrique-dev/Hardware-Price-Tracker-App.git
```

Entre no diretório do projeto

```bash
  cd hardware-price-tracker
```

Instale as dependências

```bash
  npm install
```

ou 

```bash
  pnpm install
```

Inicie o servidor

```bash
  npm run dev
```

ou 

```bash
  pnpm dev
```
